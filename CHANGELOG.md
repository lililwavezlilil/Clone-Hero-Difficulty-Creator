### ***v1.1***
*• Added note detection for Easy, Medium, and Hard difficulties to scan for 0 notes and override that difficulty with a proper chart (added due to people "charting" blank lower difficulties with stuff like global events and using other wonky auto/down charting tools)*

*• Changed the core down charting logic to create a chart from the previous difficulty (Hard gets charted from Expert, Medium gets charted from Hard, and Easy gets charted from Medium instead of all difficulties getting charted from Expert)*

*• Added a forbidden logic to find and fix any already charted Medium and Easy chart from having orange notes*

*• Added a forbidden logic to find and fix any already charted Easy chart from having blue notes*

*• Added adjustable note charting logic to all difficulties (a lower value will create less notes and more spacing while a higher value will create more notes and less spacing)*
